Será necessário a utilização da linguagem de programação R, sob a IDE, de prioridade, RStudio. Siga os seguintes passos para instalação do software em seu dispositivo.

1.1 Instalando o R

O R foi desenvolvido por Ross Ihaka e Robert Gentleman, tendo sua origem em 1993. Sua capacidade de processamento pode ser disponibilizada pela instalação do R GUI em seu dispositivo.

Em uma interface simples e pouco interativa, o R GUI é a base para que a linguagem seja rodada em seu sistema. Seu processamento é totalmente gratuito e está disponível em uma política Open-Source. Os links para instalação são os seguintes:

Windows: https://cran.r-project.org/bin/windows/base/

Mac: https://cran.r-project.org/bin/macosx/

Linux (Debian, Fedora, Ubuntu, Suse e Redhat): https://cran.r-project.org/bin/linux/

A fim de tornar a interação do interlocutor com a linguagem uma experiência mais amigável, foi desenvolvida a IDE do R no RStudio.

Para instalar o RStudio em seu dispositivo, é necessário que já esteja instalado o RGUI em sua versão necessária correspondente à atual. Para instalar o RStudio, segue os links:

Windows: https://rstudio.com/products/rstudio/download/#download

Mac: https://rstudio.com/products/rstudio/download/#download

Linux (Debian, Fedora, Ubuntu, Suse e Redhat): https://rstudio.com/products/rstudio/download/#download

1.2 Conhecendo o RStudio

Após instalar e abrir a interface em seu dispositivo, é possível visualizar quatro áreas para exploração do ambiente.

No canto superior esquerdo, como na imagem acima, é encontra-se a região “Source”. Nessa parte, é possível realizar a execução dos scripts.

No canto inferior esquerdo, é possível encontrar o console. Nele, podemos executar comando imediatamente sem registrar algo que possa ser salvo.

No canto superior direito, localiza-se o ambiente. Nele há abas que serão disponibilizados os retornos, sejam de armazenamento de variáveis, histórico de execução e as conexões.

No canto inferior direito, é possível encontrar o diretório em que o R está sendo executado no momento em seu dispositivo, plotagem de gráficos, entre outros.

Para tornar mais interativo ainda o R, é possível criar um R Notebook.

Em um Notebook, é possível realizar a execução dos scripts em um bloco de código em que, ao invés de retornar no console a resposta, o retorno será imediatamente abaixo do código. Para criar um bloco de código basta pressionar: Ctrl + Alt + I, simultaneamente. Para executar um código, pressiona-se: Ctrl + Shift + ENTER. Para executar apenas uma linha, executa-se apenas: Ctrl + ENTER.

2 Declarando a primeira variável

Trabalhar com lógica de programação, exige do autor o armazenamento de valores aos quais serão úteis para o script em questão. Imagine que você faça um script que calcula a soma de dois valores distintos e essa soma será fulcral para seus próximos passos.

9+7
## [1] 16

Perceba que, ao executar a soma de nove com sete o R nos retorna uma resposta válida e clara, 16. A fim de reutilizar esse valor, é possível armazenar essa soma em uma variável de um determinado nome. Irei armazenar a soma em uma variável “x” e retornarei o valor de x.

x <- 9+7
x
## [1] 16

De agora em diante, em meus códigos posso utilizar a variável “x” que assume a soma das constantes 9 e 7. Perceba que, para atribuir esse armazenamento foi utilizado o nome da variável, o símbolo “<-” e o valor a ser atribuído. É possível realizar o mesmo processo com o sinal de “=”. Entretanto, é necessário que seja levada uma atenção nesse caso.

Ao utilizar o sinal “=” em nosso código, lê-se a situação como: x recebe 9 + 7. Diferente do comum, não se lê: x é igual a 9 + 7. Portanto, utilize da forma que se sentir a vontade, mas fique atento com os erros.

3 Pacotes no R

No procedimento de execução de algumas tarefas, nativamente o R pode se limitar com algumas funcionalidades. Diante dessa situação, será necessária a instalação de um pacote. Mas o que são os pacotes?

Um pacote do R é um conjunto de funcionalidades disponibilizadas gratuitamente e desenvolvidas pela comunidade capaz de realizar processos que vão além do nativo. São mais de vinte mil pacotes disponíveis no repositório atual.

Para encontrar a documentação de algum pacote, é necessário acessar o seguinte link: https://cran.r-project.org/web/packages/available_packages_by_name.html. Neste, estão os pacotes oficiais do repositório CRAN.

Como utilizar um pacote?

A priori, é necessário realizar a instalação de um pacote em seu dispositivo. Para instalar, basta digitar o seguinte comando no R:

install.packages("nome_do_pacote")

O nome do pacote deve estar, obrigatoriamente, entre aspas. Caso o pacote exija dependências, é possível realizar o seguinte comando:

install.packages("nome_do_pacote", dependencies = TRUE)

Também é possível instalar um pacote indo na área do canto inferior direito na aba Packages. Nesta aba, clique em Install e instale o pacote desejado.

Para utilizar um pacote, é necessário carregá-lo no ambiente em execução. Para carregar um pacote, digite:

library(nome_do_pacote)

Caso seu objetivo seja utilizar apenas uma função do pacote, é possível utilizar o seguinte código:

nome_do_pacote::nome_da_função
